Deque

Deque是Queue的子接口，Deque有两个比较重要的类：LinkedList与ArrayQueue，建议使用栈时，用ArrayQueue的push和pop方法，可以实现和Stack相同的效果，使用队列时，使用ArrayDeque的add和remove方法，所以Deque既可以用作栈，又可以用作栈。
LinkedList即是List，又是Queue，还是Deque。但是我们在使用的时候，总是用特定的接口来引用它，这是因为持有接口说明代码的抽象层次更高，而且接口本身定义的方法代表了特定的用途。

Deque有三种用途：

+ 普通队列（一端进一端出）：Queue queue = new LinkedList()`或`Deque deque = new LinkedList()
+ 双端队列（两端都可以进出）：Deque deque = new LinkedList()
+ 堆栈：Deque deque = new LinkedList()

==注意：Java堆栈Stack类已经过时，Java官方推荐使用Deque替代Stack使用。Deque堆栈操作方法：push()、pop()、peek()。==

Deque接口扩展了 Queue 接口。在将双端队列用作队列时，将得到 FIFO（先进先出）行为。将元素添加到双端队列的末尾，从双端队列的开头移除元素。从 Queue 接口继承的方法完全等效于 Deque 方法，如下表所示：

Set接口最长用的两大实现：HashSet TreeSet

Java 集合类中的 Set.add() 方法用来向 Set 集合添加对象。如果 Set 集合中已经包含相同的对象，则不改变 Set 集合。该方法返回值为 boolean 对象，如果 Set 集合中不包含要添加的对象，则添加对象并返回 true，否则返回 false。

236. 二叉树的最近公共祖先:【递归与回溯】详解

首先我们需要将题目意思抽象出来：

  1. 什么是p、q的公共祖先：即p、q同时存在于其左右子树中
  2. 什么是最近公共祖先：即该节点是p、q的公共祖先且其左右节点不是p、q公共祖先
  3. 根据上两点我们可以得知：设公共祖先是r，则有以下几种情况：
    • p、q是其左右节点
    • p或q就是r，且另一个节点在其子树中

找公共祖先，首先想到自底向上，在二叉树中，后序遍历是天然的自底向上的回溯算法，所以我们可以利用后序遍历，先处理左右子树，然后自底向上返回。
